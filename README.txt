Retention – один из самых важных показателей в компании. Есть задача – написать функцию, которая будет считать retention игроков (по дням от даты регистрации игрока). Данные имеют следующую структуру:

problem1-reg_data.csv – данные о времени регистрации, датафрейм с двумя колонками: первая - TS с датой регистрации, вторая уникальный ID.

problem1-auth_data.csv – данные о времени захода пользователей в игру, датафрейм с двумя колонками: первая - TS с датой захода, вторая уникальный ID

Написал две функции: первая берет на вход дату и расчитывает retention по последующим дням, а также строит график с первого по 365 день, вторая функция берет дату начала периода и дату конца периода, а также день, для которого нужно посчитать RR, выводит график изменения по датам, а также топ10 дат с максимальным показателем retention на указанный день.
